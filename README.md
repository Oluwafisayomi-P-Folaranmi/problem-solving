# Problem Solving

This is a repository for some problem solving questions that I have tried in the past.

I have provided the solution code in Java, while the **problem-solving process** that I followed are available on request.

A folder holds the solution to a problem. For code reusability and better arangement, a class is used to package the solution using utility methods. For example, in the chapter for **`0x01-Solving_Pure_Puzzles`** is a folder that holds normal and random common programming problems. One of the projects under this set is one that solves how to write a sideways triangle pattern which is common during output processing. I refer to this as **A Sideways Triangle**. This problem solution is packaged in the project folder **`A_SIDEWAYS_TRIANGLE-project`**. In this folder exists a class named **`A_SIDEWAYS_TRIANGLE_CLASS`**. This class for its purpose holds a member method that has the solution to write a sideways triangle.

Apart from common programming problems, the project contains problems ranging from arrays; the fundamental dynamic data structures, linked lists; using classes as in object-oriented programming; working with recursion; stacks; queues.

As said in the beginning, the problem process that I have come up with and also referenced from other sources are documented and available on official request.

Thank you. I hope you enjoy the problems.
